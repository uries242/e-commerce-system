## PROJECT
TypeScript and Advanced JavaScript
This week's challenge was to create a functional, real-world E-commerce Product Management System using TypeScript. The challenge project outlined specific steps in order to deliver the final product. As stipulated, the application uses data derived from the Dummy JSON Products API located at https://dummyjson.com/docs/products. The developer will research and analyze the Dummy JSON Products API to implement features that interact with this API.


## CHALLENGES
How I implemented TypeScript features & OOP principles, the challenges I encountered, how I overcame them and how I handled asynchronous operations with error management?
This project presented minor challenges involving setting up the specified folder structure, structuring the code with proper imports and exports, as well as resolving the errors caused by the CommonJS and ESM module systems. As it specifically relates to coding, I encountered challenges with implementing async/await with proper error handling and creating the custom error classes. Making the return types consistent across the utility functions also required additional refactoring and multiple iterations. These challenges were resolved by referring to the lecture notes, reviewing relevant documentation found on MDN and W3 Schools and using Google search for templates.
